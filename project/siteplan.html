GreenEdge Landscape

Site Plan Document

Site Name

GreenEdge Landscape

This name reflects a modern, eco-friendly, and professional landscape company that helps clients achieve a clean and vibrant edge in their outdoor spaces. It’s memorable, simple, and speaks to both greenery and design precision.

Domain name: greenedgelandscape.com

Site Purpose

This site is intended to promote GreenEdge Landscape’s services and provide customers with an easy way to learn about landscaping options, browse project galleries, and request a quote for residential or commercial landscaping projects.

Scenarios

  • What landscaping packages does GreenEdge Landscape offer for small home gardens?
  • How can I request a free quote or consultation?

Color Schema

The website will use the following color scheme:

  • Dark Green (#2e7d32): Used for headings, navigation bar, and buttons.
  • Light Green (#66bb6a): Used for accent sections, hover effects, and call-to-action elements.
  • Off-white (#f2f9f4): Used as the general background color of the website for a clean, natural look.

Typography

The website will use the following fonts:

  • Segoe UI: Used for all headings. It gives a clean and modern feel.
  • Verdana: Used for body text due to its readability and compatibility with most devices.
